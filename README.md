rfriends_userlandはandroidのuserland環境でrfriends3を動作させるスクリプトです。  
  
cd ~/  
sudo apt install git  
git clone https://github.com/rfriends/rfriends_userland.git  
cd rfriends_userland  
sh rfriends3_userland.sh
